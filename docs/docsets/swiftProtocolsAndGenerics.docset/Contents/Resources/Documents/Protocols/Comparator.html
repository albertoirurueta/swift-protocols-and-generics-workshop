Protocolo para permitir comparar objetos de tipo T.

  • compare(_:_:)

    Compara dos objetos de tipo T. - parameter o1: primer objeto a comparar. - parameter o2: segundo objeto a comparar. - returns: -1 si o1 es menor que o2, 0 si o1 y o2 son iguales o 1 si o1 es mayor que o2.
